# Largest Number Problem

Given a list of non-negative integers, arrange them such that they form the largest number.

## How to Run
Run the Python script:

```bash
python largest_number.py
```

## Example
**Input:**
```
n = 4
nums = [3, 30, 34, 5]
```

**Output:**
```
Largest number: 534330
```
